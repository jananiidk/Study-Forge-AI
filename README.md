# StudyForge AI

AI-powered study assistant built using:

- React (Frontend)
- Node.js + Express (Backend)
- FastAPI (AI Service)
- Ollama
- ChromaDB
- PyMuPDF

## Features

- PDF Upload
- AI Summary Generation
- AI Question Answering
- MCQ Quiz Generation
- Vector Search using ChromaDB

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express

AI Service:
- FastAPI
- Ollama
- ChromaDB
- PyMuPDF
